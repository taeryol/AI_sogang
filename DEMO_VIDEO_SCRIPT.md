### 🎭 SCENE 5: 기술 하이라이트 (7:30 - 8:30)

- 기술 스택 인포그래픽:

```
┌─────────────────────────────────────┐
│         Frontend                    │
│  TailwindCSS + Vanilla JS           │
├─────────────────────────────────────┤
│         Backend                     │
│  Hono Framework (TypeScript)        │
│  Cloudflare Pages/Workers           │
├─────────────────────────────────────┤
│         AI Layer                    │
│  • GPT-4 (답변 생성)                │
│  • GPT-3.5 Turbo (쿼리 최적화)      │
│  • text-embedding-3-small (벡터화) │
├─────────────────────────────────────┤
│         Database                    │
│  • Cloudflare D1 (메타데이터)       │
│  • Pinecone (벡터 검색)             │
└─────────────────────────────────────┘
```

**[내레이션]**:
> "**핵심 기술 1: Query Reformulation**  
> 자연어 질문을 AI가 검색에 최적화된 키워드로 변환합니다.
> 
> **핵심 기술 2: Hybrid Search**  
> 벡터 검색과 키워드 검색을 결합하여 정확도를 높였습니다.
> 
> **핵심 기술 3: RAG (Retrieval Augmented Generation)**  
> 검색된 문서를 기반으로 신뢰할 수 있는 답변을 생성합니다.
> 
> 그리고 모든 것이 Cloudflare의 글로벌 엣지 네트워크에서
